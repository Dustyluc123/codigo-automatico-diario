
[DOC] - Tarefa Concluída:### Frontend Development Roadmap

**Sprint 1: Estrutura Básica e Estilização Inicial**

-   [x] No `index.html`, adicione links de navegação de exemplo (ex: Home, Produtos, Contato) dentro da tag `<nav>`, utilizando uma estrutura `<ul>` e `<li>`. Em seguida, no `style.css`, aplique estilos básicos ao `body` para definir a tipografia global (font-family, font-size). Por fim, estilize os links de navegação (`nav a`) para remover o sublinhado, definir uma cor e adicionar espaçamento. **CONCLUÍDA**
-   [ ] No `index.html`, adicione um elemento `<header>` acima da navegação com um título `<h1>` (ex: "Meu Site Incrível") e um subtítulo `<p>`. Em seguida, no `style.css`, estilize o cabeçalho (`header`) para ter um `text-align: center`, um `background-color` e `padding` para que se destaque.
-   [ ] Crie um `<main>` no `index.html` e adicione uma seção de "Sobre Nós" com um título `<h2>` e alguns parágrafos de texto. No `style.css`, adicione um `max-width` e `margin: auto` à seção para centralizar o conteúdo e um `padding` para espaçamento interno.
-   [ ] No `index.html`, adicione um rodapé `<footer>` com informações de copyright. No `style.css`, estilize o rodapé para ter um `background-color`, `text-align: center` e `padding`.

**Sprint 2: Responsividade e Componentes**
-   [ ] Implementar meta viewport no `index.html` para responsividade.
-   [ ] Adicionar media queries no `style.css` para ajustar a navegação em telas menores (e.g., stacked navigation).
-   [ ] Criar um formulário de contato simples no `index.html` dentro de uma nova seção, com campos para nome, email e mensagem.
-   [ ] Estilizar o formulário de contato no `style.css`, incluindo labels, inputs e um botão de envio.

**Sprint 3: Interatividade (JavaScript)**
-   [ ] Adicionar um arquivo `script.js` e linká-lo no `index.html`.
-   [ ] Implementar uma funcionalidade JavaScript simples, como um "scroll-to-top" button ou um contador de cliques.
-   [ ] Validar o formulário de contato usando JavaScript (ex: verificar se campos obrigatórios estão preenchidos).
