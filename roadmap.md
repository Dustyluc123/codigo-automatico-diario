### Frontend

*   **TAREFA 1: Estrutura HTML Básica**
    *   Crie o arquivo `index.html` com a estrutura básica (doctype, html, head, body).
    *   Adicione um cabeçalho (`<header>`), um conteúdo principal (`<main>`) e um rodapé (`<footer>`).
    *   Dentro do `<main>`, crie uma seção para exibir produtos (`<section class="product-grid">`).
    *   Adicione pelo menos 3 cards de produto (`<div class="product-card">`) com imagem, título, descrição e preço.
    *   *Status: CONCLUÍDA*

*   **TAREFA 2: Estilização CSS Inicial**
    *   Crie um novo arquivo chamado `style.css` na raiz do projeto. Mova quaisquer estilos inline que estejam no `index.html` para este novo arquivo. Adicione regras de CSS básicas para estilizar o cabeçalho (`header`), o rodapé (`footer`), e a grade de produtos (`.product-grid` e `.product-card`). Finalmente, vincule este novo arquivo `style.css` ao `index.html` usando a tag `<link>` dentro do `<head>`.
    *   *Status: CONCLUÍDA*

*   **PRÓXIMA TAREFA: Responsividade e Interatividade Básica**
    *   Implemente um layout responsivo para a grade de produtos (`.product-grid`) usando media queries para diferentes tamanhos de tela (ex: mobile, tablet, desktop).
    *   Adicione um efeito hover básico aos `.product-card` para melhorar a interatividade do usuário (ex: um leve aumento de escala ou sombra).
