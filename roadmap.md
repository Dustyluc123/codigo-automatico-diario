
[DOC] - Tarefa Concluída:## Frontend Development Roadmap

1.  Crie o arquivo `style.css`. **[CONCLUÍDA]**
2.  Crie o arquivo `index.html`. Adicione a estrutura HTML básica (doctype, html, head, body) e vincule o `style.css`.
3.  Crie o arquivo `script.js`.
4.  Adicione um cabeçalho (header) ao `index.html` com um título.
5.  Crie a seção principal do `index.html` para exibir produtos.
6.  Crie um rodapé (footer) para o `index.html`.
7.  Desenvolva os estilos CSS para o cabeçalho.
8.  Desenvolva os estilos CSS para o rodapé.
9.  Desenvolva os estilos CSS para os cards de produto.
10. Implemente a lógica JavaScript para buscar e exibir produtos.
11. Implemente a funcionalidade de adicionar produtos ao carrinho (opcional, pode ser uma próxima fase).
