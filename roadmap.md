
[DOC] - Tarefa Concluída:## Roadmap do Projeto

Este roadmap descreve as principais etapas e tarefas para o desenvolvimento do nosso projeto web.

### Fase 1: Estrutura Base e Estilização Inicial

- [CONCLUÍDA] Criar o arquivo `index.html` com a estrutura básica de cabeçalho, navegação, conteúdo principal e rodapé.
- [CONCLUÍDA] Criar o arquivo `style.css` com o CSS essencial para o layout geral, incluindo o estilo do cabeçalho e do menu de navegação. Adicionar media queries para garantir a responsividade em telas de desktop e mobile. O menu em telas menores deve se transformar em um ícone de 'hambúrguer', que será funcionalizado com JavaScript posteriormente.
- [PRÓXIMA] Criar o arquivo `script.js` e implementar a funcionalidade para o menu de hambúrguer, permitindo que ele alterne a visibilidade do menu de navegação em telas mobile.
